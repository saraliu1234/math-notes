\begin{definition}
A function $f : \mathbb{R}^n\rightarrow \mathbb{R}$ deﬁned on a convex set $X \subset \mathbb{R}^n$ is said to be convex, if the following inequality holds true:
\[
f(\lambda x_1 + (1-\lambda) x_2)\leq \lambda f(x_1)+(1-\lambda)f(x_2)
\]
for all $x_1,x_2\in X$, and all $\lambda \in [0, 1]$.
\end{definition}
